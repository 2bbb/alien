What is Articial Life Environment (ALiEn) ?

ALiEn is a GPU-accelerated physics engine with focus on artificial life computations. It provides an abstraction of a physical world in 2D and offers the capability to programm matter building blocks. Each simulated machine (named cell cluster) has an internal structure of building blocks (named cells) modeled and visualized as a graph. They can perform both physical actions and information processing by circulating tokens along the nodes.

First steps

Playing with existing simulations

The easiest way to get to know the ALiEn simulator is to load and run an existing example file. You find various demos in examples/* demonstrating capabilities of the engine ranging from physics examples, building machines, replicators to small ecosystems.

To this end, go to Simulation -> Load and then on Simulation -> Run in the menu or use the toolbar. During running simulations you can get to know the navigation and monitoring functions:

  • • zooming and visualization mode: View -> Zoom in as well as View -> Zoom out
    • . after reaching a certain zoom level the view switches to edit mode.
  • • toggle monitor: View -> Monitor
By toggling again Simulation -> Run the simulation paused.

Further useful computation functions:

  • • snapshots: By clicking on Simulation -> Snapshot the current simulation state is saved in memory. You can continue running the simulation and return to the saved state by invoking Simulation -> Restore.
  • • single step forward and backward: If a simulation is paused a single time step can be calculated via Simulation -> Step forward. The special thing about this feature is that the old state is saved and you can go back one time step by Simulation -> Step backward. This also works several times in a row.
  • • display link: By disabling via View -> Display link one can disable rendering and thus accelerate the entire simulation.
  • • accelerate active clusters: It is possible to assign active clusters (cell clusters with tokens) more computing power by periodically freezing the remaining content for certain amount of time steps. This can be achieved by toggling Simulation -> Accelerate active clusters.

    Fun functions: One feature that is a lot of fun is applying forces to bodies in a running simulation. During a simulation one can press hold the left mouse button and move the cursor in a certain direction. Forces are exerted on every body on the path. By pressing the right mouse button, only rotational forces are applied.
